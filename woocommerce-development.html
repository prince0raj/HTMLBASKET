FAQ

1. What is WooCommerce?
WooCommerce is an eCommerce plugin for WordPress websites that allows you to create amazing looking online stores. This powerful selling tool is customisable and extendable. To read more about WooCommerce, see http://www.woothemes.com/woocommerce/.
2. Does WooCommerce work with my current theme?
Yes, WooCommerce works with any WordPress themes, however you might require a little extra tailoring to ensure it looks its best, which we can assist with.
3. How do I request extra features, like extensions and themes?
When you place an order with us, you can make any additional requests by selecting the ‘eCommerce implementation’ option. If you are still unsure you can contact us.
4. Why opt for WooCommerce over other solutions?
WooCommerce is extremely user friendly, and is simple to manage products, manage orders and so on all under a single user friendly dashboard/menu. You are able to create beautiful product galleries and can add unlimited product specifications, as well as use an integrated blog feature and a wide range of customisations. This eCommerce solutions looks fantastic, is user friendly and supports a wide variety of payment gateways.
5. What costs are involved?
There may be additional costs depending on what features you might want adding in, such as customised extensions and implementation of extensions. We charge generally on an hourly basis for these but we will do our best to give an estimated cost if you contact us.
6. How long will a WooCommerce implementation take?
Usually this takes a minimum of 3 working days, but this can take longer depending on project complexity, optional extras such as extensions, whether you want extensions implemented by us and so on.
7. Can you help with customisation of my site?
Yes, we have PHP developers who can provide coding to support custom features such as information sorting and filtering.
8. Do I need to understand PHP to use the service?
No, WooCommerce is designed with a user friendly interface, so once we have implemented and completed the project, it will be easy to use with no PHP knowledge.
9. Can I request a responsive WooCommerce site?
Yes, we can create a responsive site so that your WooCommerce store is cross-device and cross-platform compatible, and so that it adapts to its environment.
