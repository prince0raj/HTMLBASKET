FAQ

1. Do I need to know PHP to have Responsive WordPress site?
No, WordPress has a really user friendly dashboard style interface that requires no PHP knowledge.
2. Why should I choose a responsive site?
In this day and age people access websites with a range of different devices. A responsive site will ensure your website is accessible on tablets, laptop, desktops, smartphones and more.
3. Do you need different layout designs for a responsive site?
This is up to you. If you provide us with designs for the different device layouts, we will code your site as per the designs. If not, we will just use the standard responsive coding and layouts will still be optimised for different devices.
4. What do you charge for Responsive WordPress conversion?
Charge is $179 for the homepage and $89 for each additional inner page.
5. What are the responsive frameworks that you work with?
Responsive frameworks that we work with include We offer Twitter Bootstrap, Sass, Foundation Framework, Boiler-Plate, Less Framework and Fluid baseline grid.
6. What is your default responsive WordPress framework?
We use the responsive media queries framework as standard unless you specify requirements for use of another framework.
7. Do I need technical ability or experience for WordPress?
No, WordPress is designed to be easily usable by anyone even with no experience.
8. Do the latest versions of WordPress upgrade automatically?
No, you would need to manually install updates. Please note that upgrades may sometimes chance the plugin and custom code settings, but we can assist you if you need help.
9. How many revisions of my design do I get?
We provide 365 days of free support after the complete site is launched, and so any issues or queries will be dealt with during this time. If you change your mind about your design however, we would have to charge extra if the work has already commenced on coding.
10. Do you handle my installation?
We do indeed, we just need website URL, WordPress admin information and ftp/c-panel login information to install for you. Installation is just $20.
11. Can you help me register a domain and do you offer web hosting?
Unfortunately we do not offer these services.
